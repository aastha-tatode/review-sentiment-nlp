# AI Product Review Sentiment Analyzer

This project uses Natural Language Processing (NLP) to analyze product reviews and classify them as Positive, Neutral, or Negative.

## Features

- Text preprocessing with NLTK
- TF-IDF vectorization
- Logistic Regression sentiment model
- Hybrid sentiment detection using VADER
- Interactive web app built with Streamlit
- Sentiment distribution visualization
- Word cloud of most common words

## Run the Project

Install dependencies:

pip install -r requirements.txt

Run the Streamlit app:

streamlit run app.py
